# servisapp
🛠️ **Desktop Mechanic Service Report Managing app** 🛠️

### Description

:arrow_right: Desktop Mechanic Service Report Managing app, serves managing app for repair reports.

:trident: Techstack : **Java**

:fleur_de_lis: Build tool : **Gradle**
